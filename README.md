# Know Jacky

### 想法

有陣子在IG上看到有許多人在傳類似的遊戲連結，叫自己的朋友來答題考驗對自己的了解，就嘗試仿造屬於自己的Know Jacky。

### 技術

使用了React + typescript + styled component + context api,這次不倚賴redux toolkit了只使用context api進行資料管理。
